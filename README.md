# Authority, a package of skills for would-be authors
This project is a collection of skills that can help would-be authors use Claude Code as a writing assistant and get their ideas further along the writing process. The goal is to have skills that provide support at all stages of writing, from creation through final editing for publication.

## Why did I create this?
I had an idea for a science-fiction novel that I really liked, but I had no idea how to get my ideas organized, let alone written down. I have a background in modeling, so I am familiar with programming and using artificial intelligence. I realized I could use the skill-building capabilities in Claude Code to create skills that could help me in the places I often get stuck.

### 1.: Get started
The biggest problem I faced was the first one: how do I get this idea out of my head?

### 2.: Build out the story elements in a coordinated fashion.
A great story is a complex web of interconnected plots and arcs for the characters to follow. But you need those characters to be sufficiently developed for the reader to want to follow them through the story.

### 3.: Ensure that the writing aligns with the world that you're building
As your world becomes more and more complex and nuanced, how can you avoid introducing plot holes? How can you do justice to your characters and their journeys?

## So how does **Authority** help?
The skills included each address a specific pain point I have experienced:
- [`/new-writing-project`](/skills/new-writing-project/SKILL.md) - Has Claude Code set-up your writing project so that other tools can be utilized.
- [`/character-study`](/skills/character-study/SKILL.md) - Has Claude Code do a character study with you to help flesh out your characters, filling in details on backstory, motivation, etc..

I am putting these together as I encounter the pain points of writing, and, as you can see, I'm not very far along. But I am committed to getting my story out of my head and written down. As I see blockers and frictions that slow my progress, I'll build some skills to address them.
